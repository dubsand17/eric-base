# 项目简介（私有仓库）

这是一个“知识收集”网站。

我在 Twitter 上看到有价值的内容时，通过自建的 Chrome Extension 一键收藏，扩展会把该条推文的关键信息（文本、图片、时间、链接等）保存到我个人的 Supabase 数据库里；然后这个网站把这些内容以卡片/瀑布流的方式展示，方便我随时浏览与回顾。

## 工作流概览

1. 在 Twitter 上浏览到想收藏的内容
2. 点击浏览器工具栏的扩展图标，确认采集
3. 扩展把数据写入 Supabase（我的个人实例）
4. 网站端从 Supabase 读取并渲染展示（支持多图、放大预览）

## 组成

- 网站前端：Next.js + React + Tailwind（瀑布流布局，卡片展示，图片预览）
- 数据存储：Supabase（Postgres）
- 采集工具：Chrome Extension（解析推文并写库）

> 这是一个纯粹为自己服务的“收藏与呈现”系统，不涉及公开投稿或多用户协作。
