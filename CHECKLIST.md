## 🎯 项目亮点

1. **无需API**: 通过Chrome扩展采集，避免API限制
2. **清爽设计**: 专注于内容展示，无干扰元素
3. **真正瀑布流**: 使用CSS columns实现
4. **响应式**: 完美适配各种设备
5. **现代化**: 使用最新的技术栈
6. **免费部署**: 使用Supabase和Vercel免费服务
7. **交互体验**: 图片查看、主题切换等
8. **信息完整**: 显示发布时间和原链接

## 🚀 快速启动

```bash
# 1. 安装依赖
yarn install

# 2. 启动开发服务器
yarn dev

# 3. 访问 http://localhost:3000
```
